# Contributing

Thanks for your interest in contributing to Claude Code Command Palette!

## Development Setup

1. Clone the repository
2. Install dependencies: `pnpm install`
3. Open in VS Code
4. Press `F5` to launch the Extension Development Host

## Scripts

- `pnpm run compile` - Compile TypeScript and run linting
- `pnpm run watch` - Watch mode for development
- `pnpm run test` - Run tests
- `pnpm run lint` - Run ESLint
- `pnpm run package` - Build production VSIX

## Pull Requests

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Run `pnpm run compile` to ensure no errors
5. Run `pnpm run test` to ensure tests pass
6. Submit a pull request

## Code Style

- TypeScript strict mode enabled
- ESLint for linting
- Keep changes focused and minimal

## Reporting Issues

Open an issue on GitHub with:
- VS Code version
- Extension version
- Steps to reproduce
- Expected vs actual behavior
